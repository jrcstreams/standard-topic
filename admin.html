Evergreen Shortcuts

Evergreen shortcuts appear on every topic + custom search (not the homepage), after each topic's own shortcuts, in the global order set here. Reorder within a section with the arrows. Use Excluded on… to turn an evergreen off for specific topics — that exclusion is shared with the topic's own tab.

Shortcut Groups

Subsections that AI Shortcuts get bucketed into on a topic page. Add / rename / reorder here; assign each shortcut to a group in Shortcuts Directory. Topic pages render groups in the order shown.

Topic Assignments

Pick a topic on the left (incl. Homepage and Custom Search) to set which AI shortcuts appear on that page. Shortcuts are bucketed by their Group (set in Shortcuts Directory) on the rendered page. Use each shortcut's section dropdown to override its group just for this topic (e.g. sort the homepage's topic-specific shortcuts into Discover / Learn / Analyze). Drag inside a group to reorder; the within-group order is preserved.

Data Files

Edit any data file directly. Pick a file, edit the JSON, click Apply JSON to validate and stage your changes in this session, then Download this file (or use Export All JSON in the header) to save back to disk. Useful for bulk edits, reorderings, or fields not surfaced in the other tabs.
