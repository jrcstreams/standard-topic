Standard Topic — Admin

Local admin tool. Click Import All JSON and select every .json file from /data/ (topics, shortcuts-directory, shortcuts-assignments, external-searches, ai-models, prompt-generator). Edit through the tabs, then Export All JSON to download the updated files. Drop them back into /data/ and push to GitHub. Allow multi-file downloads when prompted by your browser.
