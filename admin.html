Standard Topic — Admin

Local admin tool. Import + export your /data/ JSON files for review and editing. Three import paths: Import All JSON for the six raw files, Import Bundle for a single combined standard-topic-bundle.json (easier to share / archive), or drop a single file in Data Files. Three export paths: Export All JSON for six separate files, Export Bundle for one combined file, or Export Topics CSV for spreadsheet review. Apply staged changes via the per-tab Apply buttons before exporting.

Edit any data file directly. Pick a file, edit the JSON, click Apply JSON to validate and stage your changes in this session, then Download this file (or use Export All JSON in the header) to save back to disk. Useful for bulk edits, reorderings, or fields not surfaced in the other tabs.
